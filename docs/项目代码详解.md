```javascript
        }

```

**脉冲圈原理：**
- `semiMinorAxis` 和 `semiMajorAxis` 是椭圆的宽高（单位：米）
- `performance.now()` = 当前毫秒数，一直在变大
- `Math.sin()` 把数值变成 -1 ~ 1 之间的正弦波
- `18 + 正弦波 * 12` 在 6 到 30 之间来回变化
- 每帧重新计算 → 圈在呼吸

## 九、常见问题

### Q: Cesium 为什么全屏？
Cesium 实例化时需要指定一个容器元素，该元素设了 `position: absolute; width: 100%; height: 100%`，撑满整个父容器。父容器是 Grid 布局的中间区域，所以地图填满中间。

### Q: 毛玻璃为什么能看到地图？
`backdrop-filter: blur(20px)` 把背后 Cesium 地图的内容模糊后透过来。如果 panel 背景不透明（`rgba` 的 alpha = 1），就看不清地图了。

### Q: 数据更新频率？
- 时钟：每秒更新
- 客流排行：每5秒重新 fetch（但实际上文件不变，数据不变）
- 脉冲圈动画：每帧（约60fps）更新
- 真实场景需要后端 API + WebSocket
